# Indian Startup Case Study

A data analysis project exploring trends, funding patterns, and sector insights across the Indian startup ecosystem using Python and pandas.

---

## Overview

This project performs exploratory data analysis (EDA) on Indian startup data to uncover meaningful patterns in funding, sectors, locations, and growth over time.

---

## Tech Stack

- **Python** — core language
- **pandas** — data manipulation and aggregation
- **matplotlib / seaborn** — visualizations
- **Jupyter Notebook** — analysis environment

---

## Getting Started

```bash

git clone https://github.com/mmuzahidulislam/Indian-Startup-Case-Study-.git
cd Indian-Startup-Case-Study

# Install dependencies
pip install pandas matplotlib seaborn jupyter

```
